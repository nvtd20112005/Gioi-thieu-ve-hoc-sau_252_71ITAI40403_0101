# README – LAB2 - Thực hành NumPy và Pandas

## 1. Công nghệ sử dụng

* Ngôn ngữ: Python 3
* Thư viện:

  * NumPy: xử lý mảng và tính toán số học
  * Pandas: đọc, xử lý và phân tích dữ liệu dạng bảng
* Môi trường: Jupyter Notebook

---

## 2. Mục tiêu bài thực hành

* Làm quen với cách sử dụng NumPy để tạo và thao tác với mảng dữ liệu.
* Sử dụng Pandas để đọc file dữ liệu và xử lý DataFrame.
* Hiểu mối liên hệ giữa NumPy array và Pandas DataFrame.

---

## 3. Nội dung chính

### 3.1 Thực hành với NumPy

**Mô tả:**

* Tạo mảng NumPy một chiều và hai chiều.
* Kiểm tra kiểu dữ liệu (dtype) và kích thước mảng (shape).
* Thực hiện một số thao tác cơ bản như reshape, truy cập phần tử.

**Kết quả:**

* Sinh viên hiểu cách tạo và thao tác với mảng NumPy.
* Nhận biết được sự khác nhau giữa các kiểu dữ liệu trong NumPy.

---

### 3.2 Chuyển đổi NumPy sang PyTorch Tensor

**Mô tả:**

* Chuyển mảng NumPy sang tensor PyTorch.
* Quan sát sự khác nhau giữa `torch.from_numpy()` và `torch.tensor()`.

**Kết quả:**

* Hiểu được trường hợp dùng chung bộ nhớ và trường hợp sao chép dữ liệu.

---

### 3.3 Thực hành với Pandas

**Mô tả:**

* Đọc dữ liệu từ file CSV.
* Xem thông tin tổng quan của dữ liệu như shape, cột, kiểu dữ liệu.
* Thao tác cơ bản với DataFrame.

**Kết quả:**

* Sinh viên biết cách đọc và kiểm tra dữ liệu bằng Pandas.
* Dữ liệu được chuẩn bị sẵn cho các bước xử lý tiếp theo.
